ByteNews is a personalized news browsing web application built using Python, Django, Bootstrap, and SQLite/PostgreSQL. 
The system allows users to select their preferred news categories and get tailored news recommendations based on their interests and reading history. 
The aim of the project is to provide a simple, fast, and user-friendly news experience.
Features:
✔ Personalized News Feed
Users can choose their preferred categories.
News is filtered based on user interests.
✔ Admin Panel
Django admin panel allows management of:
Categories
News articles
User details
✔ Search & Filter Options
Users can search for news by keywords.
Category-based filtering for easier browsing.
